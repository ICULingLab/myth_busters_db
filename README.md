# Myth Busters DB
A database system for project, "COVID-19 Myth Busters in World Languages".
It is now in an early development stage.

## Requirements
* Ruby v. 2.6
* SQLite3
* Node.js
* Yarn

## Installation
1. Download
```
    git clone https://github.com/jdkim/myth_busters_db.git
```
2. Change directory
```
    cd myth_busters_db
```
4. Install dependencies
```
    bundle install
```
5. Prepare DB
```
    rails db:migrate
```
## Deploy
```
    rails s
```
